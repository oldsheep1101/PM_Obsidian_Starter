# 欢迎来到 PM Obsidian Starter 🚀

你好，新的产品经理！欢迎使用这个专门为你打造的 Obsidian 笔记系统。

## 什么是 "第二大脑"？

"第二大脑"（Second Brain）是一个数字知识管理系统，帮助你：
- 存储和组织你的想法、见解和发现
- 建立概念之间的连接
- 将信息转化为行动
- 构建个人的知识体系

## 为什么选择 Obsidian？

Obsidian 是一个强大的知识管理工具，特别适合产品经理：

### 🎯 核心优势
- **双向链接**：建立笔记间的关联，发现隐藏的模式
- **本地存储**：数据安全，完全由你控制
- **高度可定制**：通过插件和主题打造专属工作流
- **快如闪电**：纯文本格式，响应迅速

### 📋 PM 工作流设计
这个 Vault 为你准备了：
1. **入门指南** - 基础概念和语法
2. **链接实践** - 手把手学习 Obsidian 核心功能
3. **模板库** - PRD、会议记录等标准化文档
4. **AI 实验室** - 智能连接、AI 助手

## 如何开始？

### 第一步：打开 Vault
1. 打开 Obsidian
2. 选择 "Open Folder"
3. 导航到 `PM_Obsidian_Starter` 文件夹

### 第二步：学习基础
1. 阅读 `Markdown_Basics.md` 掌握基础语法
2. 按照 `Linking_Practice.md` 的10个练习学习链接功能

### 第三步：使用模板
浏览 `02_Templates` 文件夹，使用 PM 常用文档模板。

### 第四步：启用 AI
按照 `03_AI_Lab/AI_Setup_Guide.md` 安装 AI 插件，提升效率。

## 💡 必备快捷键
- `Cmd/Ctrl + K`：创建链接
- `Cmd/Ctrl + P`：快速查找文件
- `Cmd/Ctrl + G`：打开知识图谱
- `Cmd/Ctrl + /`：命令面板
