# Markdown 基础语法 - 5分钟入门

> Markdown 是 Obsidian 的基础语法，简单易学。掌握这些，你就能高效地组织笔记。

---

## 📝 文本格式

### 基本文本
```markdown
这是普通文本
**这是粗体**
*这是斜体*
~~这是删除线~~
```

**效果：**
这是普通文本  
**这是粗体**  
*这是斜体*  
~~这是删除线~~

## 🔗 链接

### 基本链接
```markdown
[链接文字](文件名.md)
```

### Obsidian 双向链接（重点！）
```markdown
[[文件名]]          - 链接到文件
[[文件名#标题]]      - 链接到文件的特定部分
[[文件名|显示文字]]  - 自定义链接显示文字
```

**示例：**
* [[用户痛点]] - 直接链接
* [[用户痛点#登录流程复杂]] - 链接到具体部分
* [[单点登录优化方案|查看方案]] - 自定义文字

## 💡 小贴士

1. **专注内容**：Markdown 语法简单，让你专注于内容本身
2. **所见即所得**：大多数编辑器都支持实时预览
3. **兼容性强**：所有支持 Markdown 的工具都能使用
4. **学会链接**：Obsidian 的核心是 `[[ ]]` 链接语法
